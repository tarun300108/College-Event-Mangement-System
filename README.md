# College Event Management System

A modern web-based College Event Management System built with PHP and MySQL. This project helps faculty create and manage events while allowing students to register, track participation, and explore upcoming campus activities.

## Features

- Faculty registration and login
- Student registration and login
- Remember me login support
- Create, edit, and manage events
- Upload custom event images
- Automatic event status handling for past/current events
- Separate registration page for students
- Student event registration tracking
- Faculty dashboard with registration pulse insights
- Modern UI with image-rich pages

## Tech Stack

- PHP
- MySQL
- HTML
- CSS
- XAMPP

## Modules

### Faculty
- Register and login
- Create event
- Edit event
- Manage registrations
- Track event insights and status

### Student
- Register and login
- View upcoming and past events
- Register for events
- View registered events
